# hpr-airframe-stability
# High Power Rocketry Airframe Integration: Optimizing Static Stability[cite: 1]
**By Wildania Baez Polanco**[cite: 1]

### 1. Project Objective
To safely upgrade a custom 70 cm launch vehicle from mid-power to high-power propulsion systems[cite: 1]. Heavy motors shift the Center of Gravity (CG) rearward, causing dangerous stability degradation[cite: 1].

### 2. The Problem & The Solution
* **The Hazard:** An unballasted H-class motor reduces the rocket's static stability margin to a catastrophic **0.089 cal** (Safety threshold is 1.0–2.0 cal)[cite: 1].
* **The Engineering Solution:** Designed an internal, concentric nose cone ballast compartment using Fusion 360 to mechanically shift the CG forward[cite: 1].
* **The Result:** Adding an optimized 80g counterweight successfully recovered a safe flight stability margin[cite: 1].

### 3. Final Simulated Flight Performance
